# Token Access System

Token Access System adalah aplikasi berbasis Visual Basic .NET yang mendemonstrasikan mekanisme keamanan sederhana melalui validasi token atau kunci akses. Proyek ini berfokus pada logika autentikasi untuk melindungi bagian tertentu dari aplikasi agar hanya dapat diakses oleh pengguna yang memiliki kredensial yang valid.

## 🌟 Fitur Utama

- **Sistem Validasi Token:** Mengecek kecocokan input pengguna dengan kunci akses (token) yang telah ditentukan secara sistematis.
- **Manajemen Hak Akses:** Mengontrol navigasi antar form atau menu berdasarkan status verifikasi token.
- **Keamanan UI Sederhana:** Menyediakan antarmuka login atau verifikasi sebelum pengguna masuk ke area utama aplikasi.
- **Feedback Visual:** Memberikan notifikasi instan jika token yang dimasukkan salah atau sudah kedaluwarsa.

## 🛠️ Teknologi yang Digunakan

- **Bahasa Pemrograman:** Visual Basic .NET (VB.NET)
- **Framework:** .NET Framework
- **IDE:** Visual Studio

## 🧠 Konsep yang Diimplementasikan

Proyek ini menonjolkan beberapa konsep logika pemrograman:
1. **String Comparison:** Membandingkan input teks dengan variabel rahasia atau database token.
2. **Conditional Logic:** Menggunakan struktur `If...Then...Else` untuk menentukan alur program setelah proses validasi.
3. **Session Management:** Menjaga status akses selama aplikasi berjalan (opsional).
